# Memory Synthesis User Guide

## Overview

Memory Synthesis analyzes your past conversations to detect patterns, identify themes, and provide personalized recommendations. Instead of manually reviewing hundreds of messages, the system automatically:

- **Detects patterns** - "You often mention deadlines" or "You frequently discuss team dynamics"
- **Identifies themes** - Your dominant emotions, goals, relationships
- **Provides insights** - "You tend to be more creative in the morning"
- **Offers recommendations** - "Consider scheduling important work before 10 AM"

This happens across **7 psychological layers**:

1. **Narrative Core** - Your life story and identity
2. **Emotional Memory** - Feelings, triggers, moods
3. **Relational Memory** - Relationships and attachments
4. **Prospective Self** - Goals, fears, future aspirations
5. **Integration Rhythms** - Patterns and cycles
6. **Transformation** - How you change and grow
7. **Purpose Engine** - Meaning and values

## When to Use Memory Synthesis

Use Memory Synthesis when you want to:

**Understand yourself better**

- "What patterns do I have?"
- "What makes me happy/stressed?"
- "How do others affect my mood?"

**Identify blind spots**

- "What am I not seeing?"
- "What keeps coming up?"
- "Where am I stuck?"

**Plan improvements**

- "When am I most productive?"
- "What goals matter to me?"
- "How can I better manage stress?"

**Get recommendations**

- "What should I focus on?"
- "How can I improve relationships?"
- "What's blocking my growth?"

## Running an Analysis

### Step 1: Start Synthesis

1. Click **Memory Synthesis** in the sidebar
2. You'll see the **Seven Layers** visualization
3. Click **Run Analysis**

### Step 2: Choose Analysis Type

Select which aspect to analyze:

**Emotional Patterns**

- What emotions dominate your conversations?
- What triggers strong reactions?
- How stable is your emotional state?
- **Best for:** Understanding mood patterns, identifying stress triggers

**Prospective Self**

- What are your goals and aspirations?
- What fears hold you back?
- How do you see your future?
- **Best for:** Career planning, goal setting, understanding ambitions

**Relational Memory**

- How do you interact with others?
- What attachment patterns emerge?
- Who influences you most?
- **Best for:** Improving relationships, understanding dynamics

**Narrative Coherence**

- What's your life story?
- What themes connect your experiences?
- How has your identity evolved?
- **Best for:** Self-discovery, finding meaning

**Full Synthesis**

- Complete analysis across all layers
- Comprehensive insights
- Full recommendations
- **Best for:** Periodic deep dives (monthly/quarterly)

### Step 3: Set Time Range (Optional)

**All time** (default)

- Analyzes everything you've discussed
- Long-term patterns

**Last week**

- Recent patterns and shifts
- Current state

**Last month**

- Monthly trends
- Seasonal patterns

**Custom date range**

- Analyze specific periods
- Compare time ranges

### Step 4: Run the Analysis

1. Click **Start Analysis**

## Troubleshooting

### Analysis Took Too Long

**Problem:** Analysis is still running after 5 minutes
**Solution:**

- Close and try again
- Check internet connection
- Try analyzing shorter time period
- Try simpler analysis type

### No Patterns Found

**Problem:** "No patterns detected" message
**Solution:**

- You might need more conversation history
- Try analyzing a longer time period
- Have you had diverse conversations?
- Some patterns need time to emerge

### Patterns Seem Wrong

**Problem:** Detected patterns don't match reality
**Solution:**

- Click âœ— Reject to give feedback
- System learns from rejections
- Run analysis again in a few days
- Try different analysis type

### Can't Confirm Patterns

**Problem:** Confirm button not responding
**Solution:**

- Refresh the page
- Check internet connection
- Try again in a moment
- Report issue if persists

### Not Getting Recommendations

**Problem:** No recommendations appeared
**Solution:**

- Confirm at least one pattern first
- Some analyses have fewer recommendations
- Try running again
- Run Full Synthesis for more recs
