# Agent Templates User Guide

## Overview

Agent Templates are pre-configured AI agent profiles that you can discover, customize, and use immediately. Instead of building agents from scratch, templates provide a starting point with:

- **Pre-configured system prompts** - Expert-tuned instructions for specific roles
- **Personality settings** - Tone, formality, and behavioral traits
- **Capability presets** - Tools and permissions pre-selected for the agent's purpose
- **Proven patterns** - Templates tested and refined by the community

## Getting Started

### Finding Templates

1. Go to **Templates** in the sidebar
2. You'll see:
   - **Recent templates** at the top
   - **Trending templates** by community rating
   - **Categories** to browse by use case

### Browsing by Category

Templates are organized into categories:

- **Productivity** - Task management, scheduling, planning assistants
- **Analytics** - Data analysis, reporting, insights agents
- **Communication** - Writing, summarization, translation helpers
- **Creative** - Content generation, brainstorming, design assistance
- **Development** - Code review, debugging, documentation agents
- **Learning** - Tutoring, explanation, research assistance

Click any category to see all available templates.

### Searching for Templates

Use the search bar to find templates by:

- **Name**: "Code Reviewer", "Content Writer"
- **Description**: "document analysis", "team collaboration"
- **Tags**: "python", "remote-work", "budget-friendly"

Example searches:

- `code review` - Find templates that review code
- `documentation` - Find templates that help with docs
- `team lead` - Find templates for team management

## Customizing Templates

### Quick Customize

Click **Customize** on any template to:

1. **Rename the agent** - Give it a name for your use case
2. **Adjust personality**:
   - **Tone**: Formal, Professional, Casual, Friendly, Playful
   - **Formality**: Choose how formal the responses should be
3. **Select capabilities**:
   - Enable/disable tools the agent can access
   - Some templates allow customization, others are fixed
4. **Save as private** - Only you can see and use it

### Full Customization

For more control, click **Open in Editor** to modify:

- **System Prompt** - Write your own instructions
- **Instructions** - Add specific guidelines
- **Knowledge** - Provide background information
- **Constraints** - Set limits on what the agent should do
- **Personality** - Fine-tune all behavioral traits

## Creating Your Own Template

Once you've customized an agent, you can save it as a reusable template:

1. Click **Save as Template** in the agent settings
2. Fill in:
   - **Name** - What to call it
   - **Description** - What it does (help others find it)
   - **Tags** - Search keywords (#productivity #writing)
   - **Visibility** - Private (just you) or Public (everyone)
3. Click **Save**

Now your custom template is saved and ready to use anytime!

### Publishing to Community

Want to share your template with others?

1. Save your template as **Public**
2. It appears in the **Templates â†’ Community** section
3. Others can see your template, rate it, and use it
4. You earn points for templates people use

**Guidelines for public templates:**

- Write clear, helpful descriptions
- Use accurate tags for discoverability
- Test thoroughly before publishing
- Include example use cases in the description

## Best Practices

### Choosing the Right Template

1. **Match your use case** - A "Code Reviewer" won't help with writing
2. **Check the rating** - 4+ stars usually means it's reliable
3. **Read recent reviews** - See what actual users say
4. **Preview before use** - Check the system prompt matches your needs
5. **Start with trending** - Trending templates are tested and popular

### Customizing Effectively

1. **Keep it simple** - Small customizations are better than rewrites
2. **Test changes** - Try one change at a time and test
3. **Preserve the core** - Don't remove the template's core purpose
4. **Document your changes** - Note what you modified and why
5. **Save variants** - Create different versions for different needs

## Troubleshooting

### Template Not Appearing

**Problem**: A template I saved isn't showing up
**Solution**:

- Check if it's private (only visible in your profile)
- Try searching for the name
- Refresh the page

### Customization Not Saving

**Problem**: My changes to a template aren't saved
**Solution**:

- Make sure you click "Save as Template"
- Check that you have a unique name
- Try refreshing and retrying

### Agent Behaving Unexpectedly

**Problem**: The agent isn't responding as the template describes
**Solution**:

- Check the system prompt in the template details
- Try simplifying your requests
- Try a different template for comparison
- Leave a review to help improve the template

### Performance Issues

**Problem**: The template is slow to respond
**Solution**:

- Try a simpler template
- Disable unnecessary capabilities
- Reduce the length of your messages
- Check your internet connection
